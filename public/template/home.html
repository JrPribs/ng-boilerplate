ng-boilerplate

A base for front-end development with AngularJS


ng-boilerplate includes:
Application
  • RequireJS - JavaScript file and module loader
  • Bootstrap - Front-end CSS framework
  • LESS - CSS pre-processor
  • Bower - Front-end package manager
  • Gulp - Streaming build system
Testing
  • angular-mocks - Mocking for AngularJS
  • Karma - Integration test runner
  • Mocha - Testing framework
  • Chai - Super-flexible assertion library
  • Protractor - End-to-end test runner

Instructions:
  1. Install bower using sudo npm install bower -g. Then run bower in the ngbp directory to install front-end dependencies: bower install.
  2. Install gulp using sudo npm install gulp -g. Then install the gulp plugins: npm install.
  3. Run gulp to compile, minify and lint front-end assets.
  4. Run gulp debug while you are developing so that you can use the unminified version of your assets.
  5. Run gulp server if you want the default task run and your files served by a development HTTP server (includes livereload!)
